
## 简介
A-Project是一个融合了各种技术的App,主要是为了将平时学习的技术实践到项目中。
总体上包含了组件化、flutter、kotlin、mvvm、jetpack、协程等技术。
平时在闲暇之余，可以刷刷文章也可以看看视频。

## 主要功能
- [**玩 Android 开放 API**](http://www.wanandroid.com/blog/show/2)首页、项目
- 浏览文章
- 开眼视频Api-视频列表
- 视频播放、详情、评论
- 登录注册功能
- 个人信息页面
- 搜索功能：热门搜索、搜索历史
- 收藏功能：添加收藏、取消收藏
- 我的积分、积分排行榜功能
